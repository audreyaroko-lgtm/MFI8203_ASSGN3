## 📋 Overview

This repository contains comprehensive solutions to Assignment 2 covering advanced Monte Carlo methods for exotic options pricing and Greeks computation. The assignment consists of three main exercises:

1. **Asian Option Pricing with Control Variates**
2. **Barrier Options with Continuity Correction**
3. **Computing Greeks Three Ways (Delta Estimation)**
